# algospot-js

자바스크립트로 [알고스팟](https://algospot.com/) 문제를 풀어보자.

## 문제 목록

- [HELLOWORLD](https://algospot.com/judge/problem/read/HELLOWORLD)
- [MERCY](https://algospot.com/judge/problem/read/MERCY)

## 설치 및 실행

### 설치

```
npm install
```

### 실행

```
node {problem_id}/solution.js < {problem_id}/data.in
```

### 테스트

```
npm test {problem_id}
```
